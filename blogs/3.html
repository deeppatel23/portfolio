NumPY : NumPY is a Python library used for working with arrays. It also has functions for working in domain of linear algebra, Fourier transform (Fourier Transform is an important image processing tool which is used to decompose an image into its sine and cosine components), and matrices.

NumPY aims to provide an array object that is up to 50x faster that traditional Python lists. The array object in NumPY is called ndarray, it provides a lot of supporting functions that make working with ndarray very easy. We will also go through a bit of statistics using NumPY library.
